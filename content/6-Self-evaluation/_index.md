During my 12-week internship at **Amazon Web Services Vietnam (AWS Vietnam)** as an **FCJ Cloud Intern**, I had the valuable opportunity to apply the theories learned at HUTECH university into real-world scenarios.

My main assignment was building the **Internal Document Management System (DMS)**. Through this project, I gained hands-on experience with Serverless architecture (Lambda, API Gateway), storage management (S3), asynchronous processing (SQS), and identity management (Cognito). This journey not only improved my backend coding skills but also shifted my mindset on how to design secure and scalable cloud systems.

### Areas for Improvement

* **Code Optimization:** While the system met all requirements, I realized my Lambda code could be better optimized for cold starts and execution time.
* **Soft Skills:** I need to be more confident during team tech talks and improve my technical English to dive deeper into advanced whitepapers.
* **Cost Management (FinOps):** During development, I sometimes over-provisioned resources. I need to build a habit of estimating costs before deployment.
