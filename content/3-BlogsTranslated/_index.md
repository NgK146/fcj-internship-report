This section lists the technical AWS blogs that I researched, translated, and summarized during my internship:

###  [Blog 1 - Building a Cost-Effective AWS KMS Strategy for Multi-Tenant Applications](3.1-Blog1/)
This blog shares a centralized encryption key management (KMS) solution for SaaS applications serving multiple customers. The solution helps balance robust security, reasonable costs, and ease of management as the number of tenants grows, without inflating encryption expenses.

###  [Blog 2 - Automating Security Baselines for Amazon Bedrock with Infrastructure as Code](3.2-Blog2/)
This blog focuses on setting up fundamental security baselines when utilizing AWS Generative AI services (like Amazon Bedrock) automatically via Infrastructure as Code. It ensures AI data remains secure and complies with organizational security policies.

###  [Blog 3 - Building a Fully Automated AWS Cost Usage Report](3.3-Blog3/)
This article provides a guide on building a fully automated, Serverless system to send daily AWS Delta Cost Usage Reports via email. This helps the management team monitor and quickly detect any abnormal cost spikes without needing direct access to the AWS Billing Console.
