Worklog ghi lại hành trình thực tập 12 tuần tại AWS First Cloud Journey, bắt đầu từ tháng 4 năm 2026. Trong suốt chương trình, tôi đã thực hành nhiều dịch vụ AWS qua các bài lab thực tế và hoàn thành đồ án cuối kỳ xây dựng Hệ thống Quản lý Tài liệu Nội bộ (DMS) trên nền tảng điện toán đám mây AWS.

Chương trình thực tập kéo dài khoảng 3 tháng với nội dung các tuần như sau:

**Tuần 1:** [Khởi tạo tài khoản AWS, AWS Credits, Budgets và IAM cơ bản](1.1-week1/)

**Tuần 2:** [Mạng VPC — Subnets, NAT Gateway, Site-to-Site VPN](1.2-week2/)

**Tuần 3:** [Libreswan VPN, Hybrid DNS, VPC Peering và Transit Gateway](1.3-week3/)

**Tuần 4:** [AWS Backup, SMB Storage Gateway, S3 Static Website, Cross-Region Replication](1.4-week4/)

**Tuần 5:** [VM Migration, Amazon FSx Multi-AZ, CloudFront CDN, S3 Versioning](1.5-week5/)

**Tuần 6:** [Security Hub, Lambda ChatOps (tự động bật/tắt EC2), ABAC với IAM, KMS mã hóa, CloudTrail, Athena](1.6-week6/)

**Tuần 7:** [RDS với Node.js, pipeline Kinesis/Glue/Athena, DynamoDB CRUD, OpenSearch + Bedrock Zero-ETL, DynamoDB Streams & Kinesis Lambda](1.7-week7/)

**Tuần 8:** [Phân tích chi phí AWS với Athena, DynamoDB với Python (boto3), Data Lake Airbnb, Kinesis/EMR/Redshift, QuickSight dashboard](1.8-week8/)

**Tuần 9:** [Nghiên cứu đồ án — Phân tích yêu cầu hệ thống DMS và thiết kế kiến trúc AWS](1.9-week9/)

**Tuần 10:** [Phát triển giao diện Frontend, Lambda backend (upload/download, Presigned URL, DynamoDB)](1.10-week10/)

**Tuần 11:** [Triển khai đầy đủ Backend AWS — CloudFront, Cognito, SQS, GuardDuty, CloudWatch, SNS](1.11-week11/)

**Tuần 12:** [Kiểm thử toàn hệ thống, sửa lỗi, hoàn thiện tài liệu và nộp đồ án](1.12-week12/)
